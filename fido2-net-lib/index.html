    # fido2-net-lib

    Hello, .net developer! Welcome to the documentation for the open source Fido2 .net library, which brings WebAuthn to the .net world!

    Please visit the Github repository (opens new window) and checkout the readme.

    # Purpose of the library

    Enable passwordless sign in for all .net apps (asp, core, native).

    To provide a developer friendly and well tested .NET (opens new window) FIDO2 Server (opens new window) / WebAuthn relying party (opens new window) library for the easy validation of registration (opens new window) (attestation (opens new window)) and authentication (opens new window) (assertion (opens new window)) of FIDO2 (opens new window) / WebAuthn (opens new window) credentials, in order to increase the adoption of the technology, ultimately defeating phishing attacks.
